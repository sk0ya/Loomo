> アプリ名: **Loomo**（ルート名前空間: sk0ya.Loomo）
> 作成日: 2026-05-31 / 最終更新: 2026-06-29

Loomo はローカル開発ワークスペース向けの **AIエージェント搭載デスクトップアプリ**（C# / WPF・.NET 9）。
自然言語の指示を、AIが **ツール（Terminal / Editor / Browser 等）** を呼んで遂行する。詳細は各ファイルへ。
